# VisionAI Modular

## Kurulum

```powershell
pip install -r requirements.txt
```

## Çalıştırma

```powershell
python main.py
```

## Notlar

- `q` tuşuna basınca kamera kapanır ve rapor penceresi açılır.
- `sounds/alarm.wav` dosyasını sen eklemelisin.
- Göz kırpmalar sayılmaz.
- Eyes Closed sadece 3 saniye aralıksız göz kapalı kalınca 1 kez sayılır.
